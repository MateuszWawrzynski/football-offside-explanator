HOW TO USE IT?

  • Click with the LEFT MOUSE BUTTON on player to change his position
  • Aim on other player and press "E" key to pass the ball
  • Aim on any player and press LEFT or RIGHT ARROW to make player run
    (Use "S" key to pause/unpause all players runs)
  • Press "F" key to toggle the offside line visibility
  • Press "R" key to reset all players positions

Reload the page (F5) if content doesn't fit to the browser window
